# ZenSmile Dental Clinic - Mobile-First Refactor

A premium, modern, and fully responsive website for ZenSmile Dental Clinic, Colaba, Mumbai.

## ✨ Features
- **Mobile-First Design**: Built from the ground up for flawless appearance on phones.
- **Glassmorphism UI**: Modern aesthetic using CSS backdrop-filters and semi-transparent cards.
- **Micro-Animations**: Scroll reveal animations and 3D hover effects.
- **Performance Optimized**: Optimized assets and lightweight CSS/JS.
- **ZenBot AI**: Integrated assistant for booking and FAQs.

## 🛠️ Testing Mobile Responsiveness
1. Open `index.html` in any modern web browser.
2. Press `F12` to open Developer Tools.
3. Click the **Device Toolbar** button (Ctrl+Shift+M) to toggle device emulation.
4. Select various devices (iPhone, Pixel, iPad) to verify the layout remains flawless.

## 🚀 How to Run
Since this is a static site, you can run it directly:
- Open `index.html` in your browser.
- **Or host it for free**: This project is ready for deployment on Render, Netlify, or GitHub Pages.

## 📁 Folder Structure
- `index.html`: Main landing page
- `about.html`, `services.html`, `gallery.html`, `contact.html`: Internal pages
- `styles.css`: Mobile-first styling system
- `main.js`: Core UX logic (animations, menu, progress bar)
- `zenbot.js`: AI Chatbot assistant
- `*.png`: Optimized visual assets
